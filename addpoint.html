добавление очков

Для начала создадим переменную «count = 0», позже добавим шрифт, 
myfont = pygame.font.SysFont('monospace', 15)
а после этого сделаем само отображение очков. 
string = myfont.render('Очков: ' + str(count), 0, (255,0,0))
Его мы должны записать перед командой покраски заднего фона. И наконец мы пишем под командой заливки заднего фона данную команду «screen.blit(string, (0, 50))». Строку с очками мы сделали, но нужно сделать так, чтобы они добавлялись.
Для этого под командой где мы импортировали библиотку пропишем очень сложную формулу: 
def inter(x1, y1, x2, y2, db1, db2):
    if x1 > x2 - db1 and x1 < x2 + db2 and y1 < y2 + db2: 
        return 1 
    else: 
        return 0
И после биндов клавиш прописываем эту команду: 
if inter(a_x, a_y, z_x, z_y, 20, 40):
        count += 1
        strike = False
        a_y = 1000
        a_x = 1000
