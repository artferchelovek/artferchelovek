создание противника

Для противника тоже нудно создать переменную, например «zet». Для этого под командой где мы создавали персонажа прописать данную команду «zet = pygame.Surface( (40, 40) )» 
И также прописать для него координаты:
    z_x = 0
    z_y = 0
Чтобы он появился на экране, нужно под командой где мы красили экран, написать это: 
screen.blit(zet, (z_x, z_y) )
Чтобы заставить нашего противника двигаться, пропишем вот это в цикл: 
right = True
    
    if right:
        z_x += 0.25
        if z_x > 400:
            z_x -= 0.25
            right = False
    else:
        z_x -= 0.25
        if z_x < 0: 
            z_x +=0.25 
            right=True
