создание игрового окна

Чтобы создать наше игровое окно, нужно сначала создать файл под названием «main.py» и открыть его удобным редактором кода, в моём случае я выбрал Sublime Text 3. После этого мы должны импортировать библиотеку «pygame» в наш код, делается это написанием строчки «import pygame» и ниже «pygame.init()».

Далее нужно написать несколько простых строчек, первая из которых 
window = pygame.display.set_mode( (400, 400) )
отвечает за создание игрового окна. Если сейчас открыть файл, то у нас появиться чёрное окошко, которое сразу исчезнет.

Чтобы сделать так, чтобы окно не закрывалось, нужно прописать целый цикл, написанный ниже: 
done = False
    while done == False:
        for e in pygame.event.get():
        if e.type == pygame.QUIT:
            done = True
И если сейчас открыть файл, то окно будет статичным и не будет закрываться.
